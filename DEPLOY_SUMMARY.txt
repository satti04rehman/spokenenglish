================================================================================

PLATFORM: Railway (Backend) + Vercel (Frontend)
COST: $5/month (backend only)
TIME: 10 minutes
DIFFICULTY: Very Easy

Step 1: Create GitHub Account (free)
   → https://github.com/signup

Step 2: Push your code to GitHub
   → Use any git tool or command line

Step 3: Deploy Backend to Railway (5 min)
   → Go to https://railway.app/
   → Click "New Project" → "Deploy from GitHub"
   → Select your repo
   → Add environment variables
   → Copy backend URL

Step 4: Deploy Frontend to Vercel (5 min)
   → Go to https://vercel.com/new
   → Import your repository
   → Set VITE_API_URL to Railway URL
   → Deploy

Step 5: Test
   → Open Vercel URL in browser
   → Login: admin / admin123
   → It works!

                         ENVIRONMENT VARIABLES
================================================================================

When deploying, set these variables:

MONGODB_URI=mongodb+srv://spoken108english:spoken108english@spoken-english.4ruepyi.mongodb.net/?appName=spoken-english

JWT_SECRET=generate_random_32_chars (use Node command below)

JWT_REFRESH_SECRET=generate_random_32_chars (use Node command below)

CLIENT_URL=https://your-vercel-url.vercel.app

NODE_ENV=production

To generate secrets in terminal:
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

================================================================================
                        TROUBLESHOOTING HELP
================================================================================

Cannot Connect to Database?
   → Check MONGODB_URI in environment variables
   → Check MongoDB Atlas IP whitelist

Frontend Can't Reach Backend?
   → Check VITE_API_URL matches Railway URL
   → Check for typos in URL

Build Failed?
   → Check build logs on Vercel/Railway
   → Run locally first to test

Getting 404 Errors?
   → Check API endpoint paths
   → Verify database is connected

More help in: DEPLOY_GUIDE.md "TROUBLESHOOTING" section
